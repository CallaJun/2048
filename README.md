calla2048
===========

## Setup

Local:

```
git clone git@github.com:grant/hnplays2048.git
npm install
node app
```

Publishing:

```
git clone git@github.com:grant/hnplays2048.git
npm install
heroku create
heroku labs:enable websockets
git push heroku master
heroku open
```

Developing:

```
node app
shell shell/sass.sh
```
